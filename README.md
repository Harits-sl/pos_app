# Persyaratan Sistem
- flutter SDK, versi yang digunakan pada projek ini versi 3.19.6
- android studio
- Visual Studio Code

# instalasi
buka projek pada IDE yang digunakan,lakukan perintah 'flutter pub get' pada terminal

```sh
cd pos_app
flutter pub get
```
